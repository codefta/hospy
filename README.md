# Hospy

Find hospital and create an appointment
This aims for aido health test

## Architecture

- FE : Reactjs
- BE : Nodejs, ExpressJS

## Installation

```bash
$ npm run install:all
```

## Usage

```bash
$ npm run start:all
```
